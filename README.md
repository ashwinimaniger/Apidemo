# APIDemo Playwright Tests

This sample contains Playwright Test scripts demonstrating navigation and basic API testing (GET, POST, PUT, DELETE).

Run locally:

```powershell
npm install
npx playwright install
npm test
```

Notes:
- Tests use `reqres.in` as a sample API for POST/PUT/DELETE behavior.
- The first test navigates to https://automationexercise.com/api_list as requested.
